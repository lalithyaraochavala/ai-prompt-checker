# AI Prompt Quality Checker

A free tool to score any AI prompt across 5 dimensions and get an improved version instantly. No signup. No API key. No data sent anywhere.

**Live:** https://lalithyaraochavala.github.io/ai-prompt-checker

---

## What it scores

| Dimension | What it checks |
|---|---|
| **Clarity** | Is the task specific and unambiguous? |
| **Context** | Does it explain the situation and audience? |
| **Constraints** | Are length, tone, and exclusions defined? |
| **Success definition** | Does it describe what a good output looks like? |
| **Verification risk** | Could the output contain unverifiable claims? |

Total score: 50 points. You also get an improved version of your prompt ready to copy and use.

---

## How it works

Pure client-side JavaScript — no backend, no API calls, no data leaves your browser. Scoring is rule-based and runs instantly.
